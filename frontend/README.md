# Frontend

This folder contains the React client for the quiz platform.

## Stack

- React 19
- Vite
- React Router
- Tailwind CSS
- Lucide React icons

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

## Available scripts

- `npm run dev` - Start Vite in development mode
- `npm run build` - Build the app for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview the production build

## Notes

The UI code lives in `src/`, with reusable components in `src/ui/` and page-level screens in `src/pages/`.
